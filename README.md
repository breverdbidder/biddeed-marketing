# BidDeed.AI Marketing

Marketing assets, research, and content strategy for BidDeed.AI - the Agentic AI ecosystem for foreclosure auction intelligence.

## Repository Structure

```
biddeed-marketing/
├── research/           # Competitive analysis, market research
├── content-ideas/      # Video scripts, post concepts, hooks
├── scripts/            # Automation scripts for content distribution
├── assets/             # Images, thumbnails, brand materials
└── campaigns/          # Campaign tracking and performance
```

## Positioning

**BidDeed.AI is an Agentic AI ecosystem, NOT SaaS.**

Core differentiators:
- 12-stage pipeline (Discovery → Archive)
- ML-powered predictions (third-party probability, sale prices)
- Multi-county processing (67 FL counties, 1K-2K auctions/day)
- Defensive ROI: Avoided losses > Promised income

## Content Pillars

1. **Deal Outcomes** - Real properties, real decisions, real results
2. **Risk Avoidance** - "The $100K I didn't lose"
3. **Velocity** - "1,000 auctions while you sleep"
4. **Technical Edge** - ML, Lien Discovery, BECA verification

## Target Platforms

- YouTube Shorts (primary)
- LinkedIn (professional credibility)
- TikTok (reach expansion)
- X/Twitter (real-time auction commentary)

## Brand Voice

- Direct, no fluff
- Show don't tell
- Concrete numbers over vague claims
- "Agentic AI" terminology
